## 🎯 Three Quick Workflows

### Workflow 1: Artist Protecting Artwork (5 minutes)
```
1. Go to ENCODE tab
2. Upload artwork + logo
3. Click encode button
4. Check metrics (verify stealth)
5. Download stego image
6. Share stego; keep backup of logo
```

### Workflow 2: Verifying Ownership (2 minutes)
```
1. Go to DECODE tab
2. Upload stego image
3. Click extract button
4. View recovered proof
5. Compare with original
6. Download extracted proof
```

### Workflow 3: Comparing Two Images (3 minutes)
```
1. Go to COMPARE tab
2. Upload Image A (leaked)
3. Upload Image B (yours)
4. Click compare button
5. Get ownership verdict
6. Export evidence
```
